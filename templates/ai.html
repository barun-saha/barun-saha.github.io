Generative AI Hackathons

2023 has been the year of Generative AI, with several Large Language Models and Large Multimodal Models becoming available. I have had participated in a few hackathons to gain a first-hand experience of these novel technologies and build a small solution or two that can help make our lives better.

A list of few such GenAI solutions developed in some of the hackathons is as follows:

  • Sys2Doc: Generate documentation based on system diagrams using Gemini Pro Vision.
  • RAG2Rich: Optimize the configuration of a Retrieval-Augmented Generation System using Vertex AI and PaLM 2.
  • Slide Deck AI: Generate a slide deck on a given topic using Mistral 7B Instruction tuned. Previously, it used Llama 2, and had won 3rd place in the Llama 2 Hackathon with Clarifai.
  • Poem2Pic: Generate an image based on a poem using Stable Diffusion.
